# Task A: Popcount Fast Path

Write a high-speed function to calculate the number of set bits in a 64-bit integer.

### Input Format
A single 64-bit integer.

### Output Format
An integer representing the set bit count.

### Sample Input
```
7
```

### Sample Output
```
3
```
