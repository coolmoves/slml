# slml
A Simple LML Blog And Problem Display Site.

> ### For Blogs:
> ```md
> - Create src/content/blogs/yourBlogName.md
> - use the sample file as template.
> ```
> ### For Pushing Problems:
> ```md
> - Create src/content/problems/problemName.md
> - The `code` in meta is the order of the questions ,contest id merges the problems to a specific contest.
> - Modify the src/content/data/contests.ts to include your contest. (Use the sample for reference).
> ```
